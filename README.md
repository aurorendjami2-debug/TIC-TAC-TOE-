
# Tic Tac Toe en C

##  Description
Ce projet est un jeu du morpion (Tic Tac Toe) développé en langage C.
Le jeu permet à un joueur d'affronter l'ordinateur.

##  Compilation
Pour compiler le programme, utilisez la commande suivante :

gcc main.c good_jeu.c -o good_jeu


##  Exécution

Sur Windows :
good_jeu.exe

Sur Linux :
./good_jeu
